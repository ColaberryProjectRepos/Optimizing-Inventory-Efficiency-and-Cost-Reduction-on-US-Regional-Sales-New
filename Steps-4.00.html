Methodology 4

Data Preparation:

The dataset was cleaned, merged, and transformed to ensure consistency across all variables. Columns related to dates, pricing, and warehouse codes were standardized for accurate analysis. Additionally, categorical variables were encoded by using feature engineering techniques to facilitate seamless model ingestion and analysis, ensuring robust and accurate results.


Model Selection:


The project initially considered time-series analysis but transitioned to a classification model using RandomForestClassifier, achieving 99.6% accuracy in predicting optimal inventory levels. This model was optimal for classifying inventory management outcomes. The RandomForestClassifier was trained on historical order data to predict the optimal order quantity for each warehouse. It achieved an accuracy of 99.6%, with an F1 score, precision, and recall also at 99.6%. The model performed well across all inventory categories, ensuring precise cost optimizations.

Step Name: Line and Stacked Column Visual (Visual 1) 19



Line and Stacked Column Visual 1(Year|Qtr|Month)

Steps
  1. Click on an empty space on the canvas, and from the visualizations pane select Line and Stacked Column Chart
  2. Fill up figures provided below for the necessary fields
Build Visual Tab:

  • X axis = Date Field (Drag the date from the Calendar Table and under the date Hierarchy unselect Day) It will enable Drill down capability automatically.
  • Column y axis=TotalOrders 
  • Line Values = GPM%, Inventory Turnover Ratio, Lead-time%, Stockout Rate
  • Tooltips = OrderQunatity, Total Profit, Total Sales, GPM%, Inventory Turnover Ratio, Lead-time%, Stockout Rate

Format Visual Tab:

General: 
  • Properties, Size, Height=330, Width=548, Position, Horizontal=204, Vertical=45
  •  Title=On, Text=Year | Qtr | Month, Font=DIN(12), Text color=#252423, Background color=#CCEED2, Horizontal alignment=Center
  •  Effects: Background=On, Background color=#FFFFFF
  • Tool Tips=on, Text=Segoe UI=10 label color, value color, Drill text and icon color=#FFFFFF
  • Background Color=#B55058

Visual: 
  • X-axis: Type=Continuous, Values: Font=Segoe UI (9), Font color=#605E5C
  • Title = Off
  •  Y-axis: Range=Min=0, Max=auto, Values: Font=Segoe UI (9), Font color=#605E5C, Display         Units=Auto
  •  Title = Off
  •  Secondary Y-axis: Values: Font=Segoe UI 9), Font color=#333333, Display Units=Auto
  •  Title = Off
  • Legend =ON  
  • Position= top left
  • Gridlines: Horizontal=OFF
  • Columns: Color=#0D6ABF, Transparency=14%
  •  Lines: Colors: GPM%=#E66C37, Inventory Turnover Ratio=#A1343C, Lead-time%=#12239E, Stockout Rate=#D9B300
  •  Markers= Off     
  • Data labels:  = Off
